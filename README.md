# goit-web-hw-11
## 🚀 Швидкий запуск проекту

### 1. Запуск бази даних у Docker
Переконайтеся, що Docker Desktop запущений. У терміналі в корені проекту виконайте команду для підняття контейнера з PostgreSQL:
```bash
docker-compose up -d
```
*Контейнер автоматично налаштує базу даних на порті `5445` у довіреному режимі.*

### 2. Встановлення залежностей додатку
Створіть віртуальне середовище (за бажанням) та встановіть необхідні Python-пакети:
```bash
pip install fastapi uvicorn sqlalchemy psycopg2-binary pydantic[email] python-dotenv
```

### 3. Запуск FastAPI додатку
Запустіть локальний сервер розробки за допомогою Uvicorn:
```bash
uvicorn main:app --reload
```
*При першому запуску SQLAlchemy автоматично створить усі необхідні таблиці в базі даних.*

### 4. Документація API
Після успішного запуску сервера перейдіть за посиланням для тестування ендпоінтів через інтерактивну документацію:
* **Swagger UI:** (http://127.0.0.1:8000/docs)
